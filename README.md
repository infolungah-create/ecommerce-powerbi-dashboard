# E-commerce Power BI Dashboard

## Project Overview
This project focuses on analyzing e-commerce sales and customer behavior using Power BI.

The dashboard was developed to provide business insights into:
- Sales performance
- Customer behavior
- Revenue trends
- Product performance
- Customer segmentation

The project supports data-driven decision-making through interactive visualizations and KPI monitoring.

---

## Tools Used
- Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Excel / CSV Dataset

---

## Dashboard Features

### Sales Analysis
- Total Revenue
- Total Orders
- Average Order Value
- Sales Trends

### Customer Analysis
- Customer Segmentation
- Returning vs New Customers
- Customer Spending Behavior

### Product Analysis
- Product Category Performance
- Top-Selling Products
- Revenue by Category

### Operational Analysis
- Delivery Performance
- Customer Ratings
- Discount Impact

---

## Key Insights
- High-value customers contributed most revenue
- Discounts influenced purchasing behavior
- Delivery performance affected customer satisfaction
- Certain product categories outperformed others

## How to Open the Project

1. Install Power BI Desktop
2. Open the `.pbix` file
3. Refresh data if required

---

## Dataset Location

Ensure the dataset file is stored in the correct location linked to the Power BI project.

If the dataset path changes:
- Open Power Query
- Update the data source path
- Refresh the dashboard

The purpose of this project is to demonstrate business intelligence and data visualization skills using Power BI to solve real-world business problems.
